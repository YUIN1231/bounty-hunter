# Easygo Mini Hackathon — Submission

## Project Name
Bounty Hunter — Enterprise Prize & Bounty Intelligence Dashboard

## Tagline
The missing tool for developer-focused enterprises: know every prize opportunity before your competitors do.

## Enterprise angle
Companies sponsoring hackathons want winners who know the platform. Bounty Hunter is the tool that sophisticated enterprise dev teams use to stay ahead — tracking not just hackathons but bug bounties, dev bounties, and ML competitions across all major platforms.

## What it does
- Aggregates opportunities from Devpost, HackerOne, Gitcoin, Kaggle, HeroX
- AI scoring engine prioritizes by ROI (prize/effort ratio)
- Real-time dashboard: filter by category, prize floor, deadline
- API-first: enterprise teams can integrate into existing tooling

## Enterprise use cases
- Dev team leads: identify hackathons to send engineers to
- Security teams: track which bug bounty programs are highest value
- ML teams: never miss a Kaggle competition announcement

## Tech Stack
Next.js 16 · TypeScript · Claude AI · REST APIs · Tailwind CSS
