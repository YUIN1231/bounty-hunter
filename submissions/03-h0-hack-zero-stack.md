# H0: Hack the Zero Stack — Submission

## Project Name
Bounty Hunter — Full-Stack Prize Discovery Platform

## Tagline
From idea to deployed web app: a full-stack bounty hunting dashboard built with modern web tooling.

## What it does
A full-stack Next.js web application that aggregates prize opportunities from 5 platforms, scores them with AI, and presents them in a real-time dashboard — built entirely with the zero-stack philosophy.

## Stack
- **Frontend**: Next.js 16 App Router + Tailwind CSS v4
- **Backend**: Next.js API Routes (serverless-ready)
- **AI**: Claude Haiku for intelligent scoring
- **Data**: JSON-based storage (zero database overhead)
- **Deployment**: Vercel-compatible (one command deploy)

## Zero Stack Philosophy Applied
- No database — flat JSON files, fast reads
- No backend server — Next.js API routes handle everything
- No auth complexity — open dashboard, local-first
- No build configuration — works out of the box

## Features
- Real-time scan of 5 platforms simultaneously
- AI-powered ranking (0-100 score per opportunity)
- Filter by category, prize amount, deadline
- Sub-5-second full scan

## Demo
Live at localhost:3005 — scannable, filterable, real data
