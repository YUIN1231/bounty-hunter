# Chutes Hack Malaysia — Submission

## Project Name
Bounty Hunter — AI Web Scraping + Intelligence Dashboard

## Tagline
5 platforms. 1 scan. AI-ranked results in under 5 seconds.

## What it does
A web application that aggregates prize opportunities from across the internet using real API integrations and AI-powered scoring.

## Technical highlights
- Parallel async scraping of 5 platforms (no blocking)
- Claude AI evaluates each opportunity in a single batch call
- Results cached to JSON, served via Next.js API routes
- Clean Tailwind dashboard with live filtering

## Why Chutes / Malaysia angle
Southeast Asian developers have world-class skills but face information asymmetry — opportunities announced on US-centric platforms don't reach them in time. Bounty Hunter solves this with automated, always-on scanning.

## Demo
Scans live, returns real data from Devpost, HackerOne, Gitcoin, Kaggle, HeroX

## Tech
